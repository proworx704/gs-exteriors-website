# G&S Exteriors Website

Marketing website for G&S Exteriors — pressure washing, house washing, gutter cleaning, and window cleaning services.

## Stack

- **Frontend:** React + Vite + Tailwind CSS
- **Hosting:** Vercel

## Deployed At

- **Production:** [gs-exteriors-website.vercel.app](https://gs-exteriors-website.vercel.app)

## Project Structure

```
├── index.html           # Main page
├── favicon.svg          # Site icon
├── robots.txt           # SEO
├── sitemap.xml          # SEO
├── assets/              # Compiled JS/CSS
├── fonts/               # Web fonts (Inter, Montserrat)
├── images/              # Service photos
├── hero-bg.mp4          # Hero background video
├── hero-poster.webp     # Video poster fallback
└── gs-logo*.{png,webp}  # Logo variants
```
